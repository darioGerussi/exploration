**Description**

Calculates the total cost of  
eating out, including the tax  
and tip.

***********************

**Box Model**

(All input are given)  
Input 1: Meal Cost  
Input 2: Tax Amount  
Input 3: Tip Amount

Output: Total Bill

***********************

**Design**

Set mealCost = 44.50  
Set taxRate = 0.0675  
Set tipRate = 0.15

Set taxAmount = mealCost * taxRate  
Set tipAmount = mealCost * tipRate  
Set totalBill = mealCost + taxAmount + tipRate

Display mealCost  
Display taxAmount  
Display tipAmount  
Display totalBill
