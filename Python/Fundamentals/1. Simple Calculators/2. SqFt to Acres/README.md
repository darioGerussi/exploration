**Description**

Calculates how many acres there are  
in a given amount of square feet of  
a parcel of land.

********************************************

**Box Model**

Input 1: Square Feet

Output: Acres

********************************************

**Design**

Set sqftInOneAcre = 43560  
Prompt user for totalSqft
 
Set totalAcres = totalSqft / sqftInOneAcre

Display totalAcres
